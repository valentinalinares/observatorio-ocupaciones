# observatorio_ocupaciones_github_pages

Versión mínima del observatorio lista para publicar en GitHub Pages.

## Qué contiene

- `index.html`
- `assets/`
- `data/observatorio_data.js`
- `.nojekyll`

## Cómo publicarlo

1. Crea un repositorio nuevo en GitHub.
2. Sube el contenido de esta carpeta al repositorio.
3. En GitHub entra a `Settings > Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama principal, normalmente `main`.
6. En carpeta, elige `/ (root)`.
7. Guarda.

Con eso GitHub Pages publicará el sitio.

## Qué no contiene

- No incluye `raw`.
- No incluye notebooks.
- No incluye scripts de procesamiento.

Esta carpeta existe solo para publicación web.
